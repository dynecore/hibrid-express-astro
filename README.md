## Como usar

Clona el proyecto

```bash
  git clone https://github.com/dynecore/hibrid-express-astro
```

Dirigete al directorio del proyecto

```bash
  cd hibrid-express-astro
```

Instala las dependencias

```bash
  npm install
```
compila astro

```bash
  npm run build
```
Inicia el servidor

```bash
  npm run start
```
